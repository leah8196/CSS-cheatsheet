Basics of How to Write CSS

CSS code is made up of two main parts, a selector and a declaration block. The declaration block consists of property-value pairs. Here we will take a look at some of the main CSS selectors, the elements of the decloration block and then how it all goes together.

Selector Type Example Description
* * Applies style to the entire document. This is the easiest to override with more specific declarations.
element p Applies style to an html element type
class .class Applies style to a specific class specified by class="class"
id #id Applies style to a single html element. Each id can only be used once in each document. Specified by id=""
Basic Elements of a Declaration Black
{ } Curly brackets
font-family Property name
: colon
Helvetica Value
; Semi-colon

Putting it all together you get something that looks like this

.class {
Font-family: Helvetica;
}
